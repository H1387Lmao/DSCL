Build Discord Bots
with DSCL

A high-level language designed specifically for Discord bot development. Write clean code that transpiles directly to Python.

examples/hello.dscl 
use pkg::discord
use random

const intents = new Intents()
const prefix = "!"
const TOKEN = "MY_VERY_SECRET_TOKEN"

intents->on_messages=True

const bot = new Bot(
	prefix,
	intents
)
cmd hello(User: person) -> bot{
	this->respond("Hi%s" % person)
}

cmd roll_dice(num: min, num: max) -> bot{
	mut number = random->randint(min, max)
	this->respond("Your random number is: %s" % number)
}

bot->run(TOKEN)

Get Started

Clone the repository and install dependencies to begin compiling your DSCL scripts.

git clone https://github.com/H1387Lmao/DSCL.git
cd DSCL
pip install -r requirements.txt

Compilation

Current version defaults to compiler mode.

python main.py test.dscl -o test.py
